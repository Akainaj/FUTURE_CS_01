Task 2 — Phishing Email Detection & Awareness

Description

This task focuses on analyzing phishing emails and identifying common social engineering techniques used by attackers.

Key Features

* Identification of phishing indicators
* Email classification (Phishing)
* Risk and impact analysis
* User awareness and prevention guidelines

Types of Phishing Covered

* LinkedIn account phishing
* Amazon payment fraud
* Google Docs phishing (Bitcoin scam)
* Business Email Compromise (CEO fraud)
* Advance fee scam (Google Forms)

Files Included

* Phishing Detection & Awareness Report (PDF)
* Email screenshots

## Outcome

Developed the ability to detect phishing emails and create awareness about email-based cyber threats.
