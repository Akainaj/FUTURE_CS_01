Overview

This repository contains three cybersecurity projects completed as part of my internship. These projects demonstrate practical skills in vulnerability assessment, phishing detection, and API security analysis.

---

Task 1 — Vulnerability Assessment Report
Description:

Performed a security assessment on a live web application using passive scanning techniques.

Tools Used:

* Nmap
* OWASP ZAP
* Browser DevTools

 Key Findings:

* Missing security headers
* Information disclosure
* Insecure cookie configuration
* Unencrypted HTTP communication

Folder: `Task-1-Vulnerability-Assessment`

---

Task 2 — Phishing Email Detection & Awareness

Description:

Analyzed multiple phishing email samples and identified social engineering techniques used by attackers.

Key Learnings:

* Email spoofing detection
* Malicious link identification
* Business Email Compromise (BEC)
* Advance fee scams

Outcome:

Developed awareness guidelines to prevent phishing attacks.

Folder: `Task-2-Phishing-Detection`

---

Task 3 — API Security Risk Analysis

Description:

Tested public APIs to identify security risks and evaluate authentication mechanisms.

Tools Used:

* Postman
* Browser DevTools

Key Findings:

* Lack of authentication (in some APIs)
* No rate limiting
* Excessive data exposure
* Secure authentication enforcement (positive finding)

Folder: `Task-3-API-Security`
